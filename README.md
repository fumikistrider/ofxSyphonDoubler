ofxSyphonDoubler
====

About
-----

This addon make 2 syphon input into 1 syphon output.
It uses ofxSyphonDirectory (ofxSyphon).

Requirements
------------

openFrameworks 0.8.0 


Dependency
------------

This addon needs:

- [ofxSyphon](https://github.com/astellato/ofxSyphon)
- [ofxUI](https://github.com/rezaali/ofxUI)
- ofxXmlSettings
